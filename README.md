# MalTester Enhanced

Herramientas para analizar malware y URLs maliciosas.

## Archivos

- **MalTesterEnhanced.exe** - Analizador de archivos ejecutables .exe
- **MalTesterURL.exe** - Analizador de URLs

## Uso

### MalTesterEnhanced
```cmd
MalTesterEnhanced.exe directorio
```

### MalTesterURL
```cmd
MalTesterURL.exe urls.txt --api-key TU_API_KEY
```

## Requisitos

- Python 3.x (para compilar)
- requests
- psutil
- pyinstaller (para compilar)
